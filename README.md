# DSA_JAVA

Java practice repository for data structures and algorithms.

This repo is kept intentionally simple: it is just a collection of folders with Java classes grouped by topic. There is no Maven or Gradle setup and no IDE project metadata checked in.

## Structure

- `Arrays`
  - `Arrays1D`
  - `Arrays2D`
- `BinarySearch`
  - `BSOn1DArray`
  - `BSOn2DArray`
  - `BsOnAnswers`
- `Blind75`
- `LinkedList`
  - `SinglyLL`
  - `DoublyLL`
  - `MediumSLL`
- `Recursion`
- `Sorting`
- `Stack`
- `Strings`
- `Trees`

## Git Setup

- Source files such as `*.java` are tracked in Git.
- Generated files such as `*.class` are ignored through the root `.gitignore`.
- IDE files such as `.idea/` and macOS `.DS_Store` files are also ignored.

## Purpose

- Maintain and track consistency in DSA practice
- Practice DSA in Java topic by topic
- Keep solutions in Git without IDE clutter
